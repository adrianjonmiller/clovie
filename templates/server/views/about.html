About

This is your server-side web application built with Clovie. It combines the simplicity of static site generation with the power of a Node.js HTTP server.

Server Features

This template includes:

  • API Endpoints - REST API with collection-based database
  • Server-Side Rendering - Dynamic pages with real-time data
  • Live Development - Hot reload with WebSocket integration
  • Native HTTP Server - Fast and lightweight (Express adapter available)

Available API Endpoints

  • GET /api/status - Server status and uptime
  • GET /api/users - List all users
  • POST /api/users - Create a new user
  • GET /api/users/:id - Get user by ID

Dynamic Routes

  • /user/:id - Server-rendered user profile pages

Getting Started

To customize this server:

  • Edit templates in the views/ directory
  • Add API endpoints in clovie.config.js api array
  • Create dynamic routes in the routes array
  • Modify styles in styles/main.scss
  • Add client-side features in scripts/app.js

Example API Usage

Try these commands in your browser's console:

// Get server status
fetch('/api/status').then(r => r.json()).then(console.log)

// Create a user  
fetch('/api/users', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ name: 'John', email: 'john@example.com' })
}).then(r => r.json()).then(console.log)
